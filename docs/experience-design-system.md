# 1. Design Philosophy

## The "Quiet Universe" Principle
KWANUS OS is not a loud dashboard; it is a quiet, vast space for transformation.
- minimized noise  
- maximize clarity  
- prioritize focus  
- support stillness  

## The Ceremonial Core

- deliberate pacing  
- soft entrances

# 2. Visual Identity

## Color Palette: "The Midnight Spectrum"
A curated set of gradients and depths representing the cosmic landscape.

| Name | Hex/CSS | Usage |
| :--- | :--- | :--- |
| **Deep Space** | `#0a0b14` | Body background, grounding foundation. |
| **Neon Pulse** | `cyan-400` | Active signals, core calls to action. |
| **Stellar Bloom** | `purple-500` | Narrative accents, secondary guidance. |
| **Void Depth** | `gray-950` | Surface layers, cards, elevation. |
| **Ghost Ink** | `gray-400` | Secondary text, non-critical info. |

## Typography: "The Navigator's Script"
- **Headings**: `Inter` / `Outfit` (Bold, tracked tightly).
- **Body**: `Inter` (Regular, high line-height for air).
- **Monospace**: `JetBrains Mono` (For technical data and IDs).

---

# 3. Interaction Patterns

## Entrance & Motion
- **Fade-In**: 400ms ease-out (For initial page load).
- **Slide-Up**: 300ms cubic-bezier (For card entrances).
- **Pulse**: Sustained, slow opacity shifts for active processing states.

## Ceremonial Feedback
- **Success**: A soft cyan glow transition, never an abrupt "pop."
- **Selection**: A subtle depth-shift (elevation-increase) rather than a color change.
- **Pacing**: Deliberate delays (200-400ms) on critical transitions to prevent cognitive overwhelm.

---

# 4. Narrative Style & Tone

## The Voice
- **Direct**: No fluff.
- **Grounded**: Focused on reality and evidence.
- **Stewardly**: Guiding the user, not commanding them.
- **Mythic**: Using language like "Engines," "Universe," "Rituals," and "Flows."

## Use/Do Not Use
- **Do Use**: "Initiate analysis," "View universe state," "Confirm alignment."
- **Do Not Use**: "Submit form," "Process data," "Click here."

# 6. Accessibility & Usability

- **Contrast**: Minimum 4.5:1 for all critical text.
- **Stillness Mode**: Allow users to disable micro-animations for high-focus sessions.
- **Focus States**: High-contrast rings for keyboard navigation, respecting the "Neon Pulse" theme.

---

# 7. Layout & Spatial Rules

## The Margin of Stillness
Use generous padding (32px+) to prevent UI elements from feeling crowded.

## Elevation System
- **Level 0**: Background / Deep Space.
- **Level 1**: Dashboard Cards / Navigation.
- **Level 2**: Modals / Wizards / Critical Panels.
